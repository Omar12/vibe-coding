A collection of fun, creative vibe-coded web projects built with HTML, CSS, and JavaScript.

### üêô GitHub Commits Viewer

**File:** `githubFirstCommits.html`

A simple tool to view the most recent commits from any GitHub repository. Enter a GitHub repo URL and instantly see the latest 5 commits with links to view them on GitHub.

**Features:**

- Clean, responsive UI built with Tailwind CSS
- Fetches commits using the GitHub API
- Direct links to view each commit on GitHub

**Features:**

- Animated pizza emojis floating across a starry night sky
- Random pizza sizes and speeds for variety
- Twinkling star background
- Exit screensaver with any keypress or click
- Fullscreen-friendly design

### üé® Emoji Letter Art

**File:** `emoji-ascii.html`

Transform any text into colorful emoji-based ASCII art. Choose your emojis and watch your text come to life in a creative, shareable format.

**Features:**

- Convert text (A-Z, numbers, spaces) into emoji art
- Customizable foreground and background emojis
- Built-in emoji picker with popular options
- Horizontal or vertical layout options
- One-click copy to clipboard
- Perfect for sharing in messaging apps

## Usage

Simply open any HTML file in your web browser to run the project. No build process or dependencies required - all projects are self-contained single-file applications.
